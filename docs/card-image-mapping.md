## Image file notes

- Official images use ₹**M** (millions); app uses ₹**Cr** (crores). Visually identical layout — only the denomination label differs in the PNG.
- `force-deal-action-card.png` — official image spells "Force Deal" (not "Forced Deal"). App uses "Forced Deal".
- `double-the-rent-action-card.png` — official image says "Double the Rent", app says "Double Rent!"
- `it's-my-birthday-action-card.png` — the apostrophe in the filename is literal.
- `blue-and-green-rent-card.png` — official image says "Blue and Green", app reverses to "Green/Dark Blue" order.
- Filename typos preserved as-is: `railraod` (missing 'l'), `railraod-and-...` in wildcard files.
